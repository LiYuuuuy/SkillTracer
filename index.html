SkillTracer: Structural Failure Attribution and Refinement of Agentic Skills in Long-Horizon Web Tasks

Yuyang Li, Yiran Dou, Jie-Jing Shao, Yueming Lyu, Ivor Tsang, Haiyan Yin
CFAR & IHPC, A*STAR · National University of Singapore · Nanjing University · Nanyang Technological University

Abstract

SkillTracer is a framework for long-horizon web agents that represents reusable skills as hierarchical, attributed plan graphs. Instead of treating skills as opaque action sequences, SkillTracer enables structural failure attribution, localized repair, and verification-aware recovery during web task execution.

Method

SkillTracer decomposes agentic skills into explicit plan nodes, executable actions, component-level grounding, and state-dependent transition conditions. When execution fails, the system localizes the failure to specific graph nodes and performs targeted repair without restarting the entire task.

Experiments

We evaluate SkillTracer on long-horizon web automation tasks and show that structured skill representations improve robustness, repairability, and cross-domain generalization.
