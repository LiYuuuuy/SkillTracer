SkillTracer: Structural Failure Attribution and Refinement of Agentic Skills in Long-Horizon Web Tasks

A verification-aware framework for representing, diagnosing, and repairing reusable web-agent skills.

Yuyang Li, Yiran Dou, Jie-Jing Shao, Yueming Lyu, Ivor Tsang, Haiyan Yin
CFAR & IHPC, A*STAR · National University of Singapore · Nanjing University · Nanyang Technological University
KDD 2026

SkillTracer turns web-agent skill reuse into a verification-aware control process. It localizes execution failures to specific plan-graph nodes, repairs only the erroneous structure, and preserves valid interaction progress for robust long-horizon web automation.

Abstract

Large language model agents have shown promising capabilities in web automation, yet long-horizon tasks remain fragile because execution errors can accumulate, obscure their causes, and force agents into repeated trial-and-error. SkillTracer addresses this challenge by representing reusable web-agent skills as hierarchical, attributed plan graphs rather than opaque action sequences. Each skill exposes its internal structure, including intent, component type, executable action, semantic description, and verifiable transition conditions.

When a skill fails, SkillTracer attributes the failure to concrete graph nodes and performs localized structural repair, allowing the agent to preserve valid interaction progress instead of restarting from scratch. This design enables more robust skill reuse, failure diagnosis, and cross-domain generalization in long-horizon web environments.

Method

Structured Skill Graphs

Skills are represented as hierarchical plan graphs with explicit node attributes and state-dependent transition conditions.

Failure Attribution

Execution failures are localized to specific intents, components, actions, or semantic descriptions using verification evidence.

Localized Repair

SkillTracer repairs only the erroneous structure and resumes execution from the preserved valid prefix.

Experiments

We evaluate SkillTracer on long-horizon web automation tasks across WebArena domains. The results show that structured skill representations improve robustness, repairability, and execution stability, especially in tasks requiring precise tool understanding and state verification.
